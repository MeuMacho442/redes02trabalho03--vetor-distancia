Esse trabalho usa GUI javafx para simular visualmente o algoritmo de vetor de distancia, levando em consideração politicas de expiração de arestas para permitir readaptação dinamica da rede. 
O usuario pode se sentir a vontade para cortar uma arestas, o que significa corte de enlace. Nessa simulação , não busquei resolver o problema de contagem ao infinito
